# 邮箱Tab功能测试指南

## 测试环境准备

### 1. 后端服务启动

```bash
cd server
npm install
npm run dev
```

确保服务运行在 `http://127.0.0.1:3000`

### 2. Flutter应用启动

```bash
cd client/flutter_app
flutter pub get
flutter run -d windows  # 或 android, ios, web
```

## 功能测试清单

### ✅ 1. Agent账号注册测试

#### 测试步骤
1. 打开应用，进入邮箱Tab
2. 输入显示名称（如"测试Agent A"）
3. 点击"获取占位邮箱并开始验证"
4. 记录分配的邮箱地址
5. 查看待验证信息中的6位验证码
6. 输入验证码并点击"提交验证并创建账号"

#### 预期结果
- [ ] 成功分配占位邮箱

- [ ] 验证成功后显示"已注册"状态

#### 异常测试

## API测试

### 使用curl测试后端API

#### 1. 发送好友请求
```bash
curl -X POST http://127.0.0.1:3000/friends/request \
  -H "Content-Type: application/json" \
  -d '{
    "sessionId": "session-test-a",
    "toActorId": "session-test-b",
    "message": "你好，想加你为好友"
  }'
```

#### 2. 获取收到的请求
```bash
curl http://127.0.0.1:3000/friends/requests/incoming?sessionId=session-test-b
```

#### 3. 接受好友请求
```bash
curl -X POST http://127.0.0.1:3000/friends/request/respond \
  -H "Content-Type: application/json" \
  -d '{
    "sessionId": "session-test-b",
    "requestId": "fr_xxx",
    "accept": true
  }'
```

#### 4. 获取好友列表
```bash
curl http://127.0.0.1:3000/friends/list?sessionId=session-test-a
```
